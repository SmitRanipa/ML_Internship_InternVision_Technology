# 🚀 Major Project: Loan Prediction

Dataset:
- Kaggle Loan Approval Dataset

Goal:
Predict Loan Approval (Approved = 1, Rejected = 0)

Workflow:
- Data Cleaning & EDA
- Hypothesis Testing (T-Test, Chi-Square, ANOVA concept)
- Feature Engineering & Selection
- Logistic Regression & Decision Tree
- Model Evaluation (Accuracy, Recall, F1, ROC-AUC)
- Cross Validation & Overfitting Check
- Model Saving (.pkl using joblib)

Priority Metric:
1. Recall
2. ROC-AUC
3. Accuracy

Outcome:
Complete ML pipeline from hypothesis testing to deployment-ready model.
